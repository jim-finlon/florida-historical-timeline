# Florida Historical Timeline Project

A comprehensive collection of 302 documents covering Florida history from the pre-Columbian era to the present day, organized as an [Obsidian](https://obsidian.md/) vault with interconnected wiki-links and thematic cross-references.

## Overview

This project documents Florida's rich and diverse history through detailed markdown files, each covering a significant event, person, place, or theme. From the first humans arriving 12,000 years ago to modern theme parks and sports championships, the timeline captures the full sweep of the Sunshine State's story.

### Document Format

Each document follows a consistent structure:
- **Overview**: Summary of the topic
- **History**: Detailed historical narrative
- **Thematic sections**: Relevant subtopics
- **Key Statistics**: Table of important data
- **Timeline**: Chronological milestones
- **Sources**: Reference materials

### Naming Convention

Files are named with the format: `YEAR_Topic_Name.md`
- Example: `1971_Walt_Disney_World_Opens.md`
- Example: `1959_Cuban_Revolution_Florida_Impact.md`

## Using This Vault

### With Obsidian

1. Clone or download this repository
2. Open Obsidian
3. Select "Open folder as vault"
4. Navigate to the Florida folder
5. Browse using the Index or graph view

### Without Obsidian

All files are standard Markdown and can be read with any text editor or Markdown viewer. Wiki-links use the format `[[filename|display text]]`.
